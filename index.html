1. 字形音義：《蘭亭集序》「會稽」
《蘭亭集序》開篇提及「永和九年，歲在癸丑，暮春之初，會于會稽山陰之蘭亭」。句中「會稽」的正確讀音為何？
✓ 正確答案：C

解析： 「會稽」為古地名，根據教育部國語辭典，正確讀音為「ㄍㄨㄟˋ ㄐㄧ」。

錯誤選項分析：

  • (A) 「會」字常見讀音，但在此為古地名特殊讀音。
  • (B) 這是「會稽」的舊讀音，現已不常用。
  • (D) 「稽」字無此讀音。
2. 字形音義：《蘭亭集序》「修禊」
《蘭亭集序》中「修禊事也」的「禊」字，其字義最接近下列何者？
✓ 正確答案：B

解析： 「禊」指古代一種在水邊舉行的祭祀活動，主要目的是洗滌身體，去除不祥，故字義最接近「沐浴」。

錯誤選項分析：

  • (A) 禊事雖為祭祀活動，但其核心動作是沐浴。
  • (C) 宴飲是禊事的一部分，但非「禊」字本義。
  • (D) 祈福是禊事的目的之一，但非「禊」字本義。
3. 字形音義：《蘭亭集序》「流觴」
《蘭亭集序》中「引以為流觴曲水」的「觴」字，其字義為何？
✓ 正確答案：A

解析： 「觴」指酒杯。「流觴曲水」即將酒杯置於水上，任其漂流，停在誰面前誰就飲酒作詩。

錯誤選項分析：

  • (B) 宴席是活動的整體，非「觴」字本義。
  • (C) 歌舞是宴席的內容，非「觴」字本義。
  • (D) 詩歌是活動的產物，非「觴」字本義。
4. 字形音義：《蘭亭集序》「信可樂也」
《蘭亭集序》中「亦足以暢敘幽情，信可樂也」的「信」字，其字義最接近下列何者？
✓ 正確答案：B

解析： 「信」在此處作副詞，表示「確實、的確」。

錯誤選項分析：

  • (A) 「相信」是動詞，語法不符。
  • (C) 「誠實」是形容詞，語法不符。
  • (D) 「書信」是名詞，語法不符。
5. 字形音義：《勞山道士》「驀然而踣」
《勞山道士》中「驀然而踣」的「踣」字，其字義為何？
✓ 正確答案：A

解析： 「踣」指跌倒、仆倒。

錯誤選項分析：

  • (B) 與「跌倒」意義相反。
  • (C) 與「跌倒」意義無關。
  • (D) 「驀然」有驚訝之意，但「踣」字本身無此義。
6. 字形音義：《勞山道士》「疢毒」
《勞山道士》中「恐遭疢毒」的「疢毒」一詞，其字義最接近下列何者？
✓ 正確答案：A

解析： 「疢」指熱病，「毒」指惡疾，故「疢毒」泛指疾病。

錯誤選項分析：

  • (B) 雖有「毒」字，但「疢毒」是疾病，非毒藥。
  • (C) 疾病可能帶來災禍，但「疢毒」本義是疾病。
  • (D) 疾病會帶來痛苦，但「疢毒」本義是疾病。
7. 字形音義：《勞山道士》「吮癰舐痔」
《勞山道士》中「吮癰舐痔」一詞，用來形容王生何種心態？
✓ 正確答案：B

解析： 「吮癰舐痔」比喻卑鄙地奉承權貴，形容王生為求仙術不擇手段、趨炎附勢的心態。

錯誤選項分析：

  • (A) 王生好逸惡勞，非勤奮好學。
  • (C) 王生遇到困難即退縮，非堅忍不拔。
  • (D) 王生自私自利，非樂於助人。
8. 語文常識：《聊齋誌異》文體特色
關於蒲松齡《聊齋誌異》的文體特色，下列敘述何者正確？
✓ 正確答案：C

解析： 《聊齋誌異》是清代蒲松齡所著的文言短篇小說集，內容多為狐鬼花妖、神仙道士，藉此諷刺社會現實。

錯誤選項分析：

  • (A) 《聊齋誌異》是短篇小說，非長篇章回小說。
  • (B) 《聊齋誌異》是文言小說，非白話小說。
  • (D) 雖有筆記小說的性質，但其文學性與結構更接近文言短篇小說。
9. 語文常識：《蘭亭集序》文學史地位
王羲之《蘭亭集序》在文學史與藝術史上的地位，下列敘述何者最為恰當？
✓ 正確答案：C

解析： 《蘭亭集序》是魏晉時期山水遊記散文的代表作，其書法藝術更是被譽為「天下第一行書」。

錯誤選項分析：

  • (A) 唐代古文運動由韓愈、柳宗元等人開創。
  • (B) 《蘭亭集序》雖有哲學思考，但非玄學名篇。
  • (D) 《蘭亭集序》是散文，非唐詩。
10. 語文常識：序跋類文體比較
《蘭亭集序》與《畫菊自序》皆為「序」體文，關於兩者序文性質的比較，下列敘述何者正確？
✓ 正確答案：B

解析： 《蘭亭集序》是王羲之為蘭亭雅集所作詩集而寫的序文；《畫菊自序》是張李德和為自己的畫菊作品集所寫的序文，兩者序文性質不同。

錯誤選項分析：

  • (A) 《蘭亭集序》是為眾人詩作作序，而《畫菊自序》是為自己作品作序。
  • (C) 《蘭亭集序》是為詩集作序，非作者自序；《畫菊自序》是自序。
  • (D) 兩者皆有抒發情感與記敘事件，但側重點不同，且此選項未能點出序文性質的根本差異。

13. 作者背景：王羲之與莊子思想
王羲之在《蘭亭集序》中對莊子「一死生、齊彭殤」思想的批判，主要體現在何處？
✓ 正確答案：C

解析： 王羲之雖理解莊子「一死生、齊彭殤」的道理，但面對生命終將消逝的現實，仍無法擺脫個人的悲痛與感傷，體現了其對莊子思想的「有保留」的批判。

錯誤選項分析：

  • (A) 王羲之雖有及時行樂之情，但其悲痛之情更深。
  • (B) 王羲之的悲痛正是對生死無常的無奈，而非無需憂慮。
  • (D) 王羲之並非主張積極入世，而是對生命有限的感傷。
14. 文意理解：《勞山道士》王生心理轉折
《勞山道士》中，王生在學道過程中，其心理轉折最能反映作者何種諷刺寓意？
✓ 正確答案：C

解析： 王生好逸惡勞，只想學得仙術以圖便利，最終因心術不正而自食惡果，反映了作者對世人投機取巧、不勞而獲的諷刺。

錯誤選項分析：

  • (A) 王生並未得道成仙。
  • (B) 王生始終好逸惡勞，並未勤奮刻苦。
  • (D) 王生從頭到尾都想學仙術，只是方法不對，並非從懷疑到深信。
15. 文意理解：《蘭亭集序》情感脈絡
《蘭亭集序》的情感脈絡，由「樂」轉「痛」再轉「悲」的關鍵轉折點為何？
✓ 正確答案：D

解析： 情感由「樂」轉「痛」再轉「悲」的關鍵在於對生命有限的體悟，即「脩短隨化，終期於盡」的無奈，最終引發「豈不痛哉」的悲嘆。

錯誤選項分析：

  • (A) 這是對自然景物的描寫，尚未觸及深層情感轉折。
  • (B) 這是情感變化的原因，但非直接的轉折點。
  • (C) 這是對莊子思想的批判，但「豈不痛哉」才是悲痛的直接表達。
16. 文意理解：《畫菊自序》菊花象徵
張李德和《畫菊自序》中，「菊花」的象徵意義與女性自覺的連結，下列敘述何者最為貼切？
✓ 正確答案：B

錯誤選項分析：

  • (A) 雖有清高之意，但「傲霜獨立」的堅韌更符合女性自覺的連結。
  • (C) 菊花的象徵意義更偏向堅毅，而非傳統的溫柔婉約。
  • (D) 菊花的色彩並非此文強調的重點。
17. 跨文本比較：自然書寫與詠物自況
黃美秀《尋熊記》與張李德和《畫菊自序》在表達個人志向或理念時，其筆法差異最主要體現在何處？

18. 跨文本比較：諷刺手法
《勞山道士》與《尋熊記》在寫作手法上，何者運用了較為明顯的諷刺手法？其諷刺對象為何？

解析： 《勞山道士》透過王生好逸惡勞、投機取巧卻自食惡果的故事，明顯諷刺了世間那些不願腳踏實地、妄想不勞而獲的人。

19. 跨文本比較：序文功能
《蘭亭集序》與《畫菊自序》作為序文，其主要功能除了說明寫作緣由外，共同具備的深層功能為何？
✓ 正確答案：B

解析： 兩篇序文皆在敘述緣由的同時，融入了作者對生命無常的感嘆（《蘭亭集序》）、對藝術的執著與自我價值的肯定（《畫菊自序》）。

錯誤選項分析：

  • (A) 兩者皆非主要闡述政治抱負。
  • (C) 雖有記錄性質，但深層功能更偏向抒情與思考。
  • (D) 兩者皆非主要批判時弊。
20. 綜合判斷：四文本主旨
綜合《尋熊記》、《勞山道士》、《蘭亭集序》、《畫菊自序》四篇文本，下列關於其主旨的判斷，何者最為恰當？
✓ 正確答案：C

解析： 四篇文本雖題材各異，但共同點在於作者透過各自的書寫，表達了對生命（《蘭亭集序》）、自然（《尋熊記》）、人性（《勞山道士》）或自我價值（《畫菊自序》）的深刻思考與體悟。

錯誤選項分析：

  • (A) 《尋熊記》強調尊重自然，非人定勝天；《勞山道士》諷刺好逸惡勞。
  • (B) 《蘭亭集序》雖有悲觀，但《尋熊記》和《畫菊自序》更偏向積極。
  • (D) 僅《勞山道士》為神怪故事，且其主旨非僅娛樂。
21. 文意理解：《蘭亭集序》情感轉折的深層原因
《蘭亭集序》中，王羲之由宴飲之樂轉為對生命短暫的悲痛，其深層原因最主要為何？
✓ 正確答案：B

解析： 王羲之在享受當下之樂後，轉而思考生命的有限性，體悟到個體生命無論長短，最終都將消逝，無法與天地同壽，這種對時間無情流逝的無奈感，是其情感轉折的深層原因。

錯誤選項分析：

  • (A) 離別的不捨是表層原因，而非深層的哲學思考。
  • (C) 雖處於政治動盪時期，但文章中並未直接提及，其悲痛更偏向對生命本質的感悟。
  • (D) 王羲之對莊子思想是「有保留」的批判，他並未完全看淡生死，反而因此感到悲痛。
22. 作者背景：王羲之對莊子「一死生、齊彭殤」的態度
王羲之在《蘭亭集序》中提及「固知一死生為虛誕，齊彭殤為妄作」，這句話最能反映他對莊子思想的何種態度？
✓ 正確答案：C

解析： 王羲之理性上理解莊子「一死生、齊彭殤」的哲理，知道從宏觀角度看，生死長短並無本質差異。然而，作為一個有血有肉的個體，他仍無法擺脫對自身生命有限的悲痛與感傷，這是一種理性與情感的矛盾。

錯誤選項分析：

  • (A) 他並非完全否定，而是「固知」，表示理性上是理解的。
  • (B) 並非表面認同，而是理性理解，情感難以接受。
  • (D) 批判的對象是莊子思想中過於超脫的部分，而非社會的錯誤認知。
23. 文意理解：《勞山道士》王生學道失敗的深層原因
《勞山道士》中，王生學道失敗的深層原因，除了好逸惡勞、心術不正外，最主要還體現了何種問題？
✓ 正確答案：C

解析： 王生學道失敗的根本原因在於其追求仙術的動機不純，他並非真心求道，而是想藉由仙術來滿足自己的私慾（如變出錢財、炫耀法術），這種功利主義的心態與道家清靜無為的宗旨背道而馳。

錯誤選項分析：

  • (A) 文章並未強調王生缺乏天賦，而是其心性問題。
  • (B) 王生對仙術是深信不疑的，只是想走捷徑。
  • (D) 意志不堅定是表象，深層原因仍是動機不純。
24. 字形音義：《勞山道士》「潔持」的深層內涵
《勞山道士》中，道士對王生說：「耐苦即入道耳。今一月，止一來歸，非潔持耶？」句中「潔持」一詞，其深層內涵最接近下列何者？
✓ 正確答案：B

解析： 「潔持」在此處的深層內涵，不僅指身體的清淨，更重要的是精神上的堅守戒律，克制世俗的誘惑與個人的慾望，尤其是王生好逸惡勞、貪圖享樂的本性，這是入道的基礎。

錯誤選項分析：

  • (A) 雖有身心純潔之意，但「克制慾望」是更具體的行為要求。
  • (C) 專心修煉是結果，而非「潔持」的直接內涵。
  • (D) 勤儉節約是道家生活方式，但「潔持」更強調對慾望的控制。
25. 作者背景：黃美秀自然書寫的獨特之處
黃美秀《尋熊記》的自然書寫，相較於傳統的遊記或科普文章，其獨特之處最主要體現在何處？
✓ 正確答案：B

解析： 黃美秀的《尋熊記》不僅僅是科學報告，也不是純粹的文學創作，而是將其在野外研究臺灣黑熊的科學數據、觀察心得，與個人對生命的熱愛、對自然的關懷融為一體，形成一種兼具理性與感性的「科學人文」書寫風格。

錯誤選項分析：

  • (A) 雖有文學性，但並非其最獨特之處，許多遊記也具文學性。
  • (C) 黃美秀的書寫充滿個人情感，並非避免主觀情感。
  • (D) 雖有呼籲保育的意圖，但其筆法更偏向溫和的啟發與教育，而非強烈的批判。
26. 文意理解：《尋熊記》體現的生態倫理觀
《尋熊記》中，黃美秀對於臺灣黑熊的描寫與其研究工作，最能體現何種生態倫理觀？
✓ 正確答案：C

解析： 《蘭亭集序》是王羲之為蘭亭雅集所作詩集而寫的序文，其目的在於記錄這次盛會的歡樂，同時抒發對生命短暫的感懷。而《畫菊自序》是張李德和為自己的畫菊作品集所寫的自序，其目的在於藉由菊花的品格來比喻自身，彰顯女性在困境中堅韌不拔的自覺精神。兩者在序文性質與深層目的上各有側重。

錯誤選項分析：

  • (A) 《蘭亭集序》是為眾人詩作作序，非為他人作品作序；《畫菊自序》是自序。且說理與抒情並非主要差異。
  • (B) 《蘭亭集序》並非自序，且其目的不僅是記錄盛會，更包含深層的生命感懷。
  • (D) 《蘭亭集序》是追憶，但《畫菊自序》並非緬懷故人，而是自況。
30. 綜合判斷：四文本的共同文學價值
綜合《尋熊記》、《勞山道士》、《蘭亭集序》、《畫菊自序》四篇文本，下列關於其共同文學價值的判斷，何者最為恰當？
✓ 正確答案：B

解析： 這四篇文本雖然體裁、時代背景和主題各異，但它們共同的文學價值在於，作者都透過對具體事物或事件的描寫，超越了單純的敘事或抒情，引導讀者對生命（《蘭亭集序》）、自然（《尋熊記》）、人性（《勞山道士》）以及自我價值（《畫菊自序》）進行深層的思考與反思，具有啟發性與哲學意涵。

錯誤選項分析：

  • (A) 雖有展現社會風貌，但並非所有文本都以「人情世故」為核心，且「反思深層意義」是更為精確的共同價值。
  • (C) 雖是典範之作，但「引導讀者反思」是更為核心的文學價值。
  • (D) 僅《勞山道士》有較強的批判性，其他文本並非以批判現實為主要目的。

31. 《勞山道士》中「潔持」的深層意涵與王生的對比
《勞山道士》中，道士要求王生「潔持」，而王生最終學道失敗。下列關於「潔持」的深層意涵，以及王生行為與之對比的敘述，何者正確？ (應選 3 項)
✓ 正確答案：A, B, D

解析：

  • (A) 「潔持」在道家修煉中，是身心清淨、不受外物干擾、堅守戒律的綜合體現。
  • (B) 王生在道觀中「耐苦」不足，如「日與同儕樵蘇，苦不堪言」，正是未能做到「潔持」中的堅忍。
  • (C) 王生心存僥倖，妄圖不勞而獲，這確實是其問題，但「潔持」更側重於對慾望的克制和戒律的遵守，而非單純的專心致志。
  • (D) 王生學得穿牆術後立即回家炫耀，最終撞牆而踣，正是未能克制虛榮心與炫耀慾，違背了「潔持」的內涵。
  • (E) 「潔持」是修道的態度和心性要求，而非考驗天賦。道士更看重的是品德與毅力。

解析：

  • (A) 王羲之從廣闊的宇宙與豐富的萬物中，反觀人類生命的有限與渺小。

  • (E) 王羲之對莊子思想是「有保留」的批判，他理性上理解「一死生」的道理，但情感上無法超脫，並非完全否定其觀點。

33. 筆法差異比較
請閱讀以下兩段文字節錄，並比較黃美秀《尋熊記》與張李德和《畫菊自序》在表達個人志向或理念時的筆法差異，至少從三個面向進行分析，並舉例說明。

節錄一：《尋熊記》
「我常常在想，黑熊的生存，其實是臺灣這塊土地健康與否的指標。當我們看見黑熊在山林中自由自在地活動，那代表著牠們的棲地是完整的，食物是充足的，人類的干擾是有限的。而我的工作，幾就是盡力去理解牠們，保護牠們，讓牠們能夠繼續在這片土地上繁衍。這不僅是為了黑熊，更是為了我們自己，為了臺灣的未來。」

節錄二：《畫菊自序》
「余性愛菊，尤喜其傲霜獨立，不與百花爭豔。每當秋風蕭瑟，群芳凋零之際，唯菊獨放，其堅貞之姿，足以勵人。余雖一介女流，然志存高遠，不甘隨波逐流。故每畫一菊，必傾注心力，冀其能傳達吾輩女性之堅毅，不讓鬚眉之氣概。」
參考答案：

筆法差異分析：

1. 表達方式與視角：

  • 《尋熊記》 (黃美秀)：採用現代自然書寫的筆法，以科學研究者的視角，結合親身觀察與理性分析。她從黑熊的生存狀態切入，將其與「臺灣土地健康」的宏觀議題連結，透過對生態系統的理解，闡述其保育理念與個人使命。語言風格較為平實、客觀，帶有科普教育的意味，但字裡行間仍流露對生命的關懷。例如：「黑熊的生存，其實是臺灣這塊土地健康與否的指標。」此句即以科學觀察為基礎，提出宏觀判斷。
  • 《畫菊自序》 (張李德和)：採用傳統詠物自況的筆法，以古典文人的視角，透過象徵與比喻來表達個人志向。她將「菊花」的「傲霜獨立、堅貞不屈」等品格，與自身「志存高遠、不甘隨波逐流」的女性自覺相連結。語言風格較為典雅、含蓄，充滿古典詩文的韻味，藉由描寫物象來抒發內在情志。例如：「唯菊獨放，其堅貞之姿，足以勵人。」此句即以菊花象徵自身品格。

2. 理念傳達的側重點：

  • 《尋熊記》 (黃美秀)：其理念傳達側重於生態保育的實踐與社會責任。她強調的是「理解、保護」黑熊的具體行動，並將此行動提升到「為了臺灣的未來」的高度，具有公共性與行動導向。她的志向是透過科學研究與大眾教育，推動環境永續發展。例如：「我的工作，就是盡力去理解牠們，保護牠們，讓牠們能夠繼續在這片土地上繁衍。這不僅是為了黑熊，更是為了我們自己，為了臺灣的未來。」明確指出其行動與社會責任。
  • 《畫菊自序》 (張李德和)：其理念傳達側重於個人精神層面的自我期許與女性價值的彰顯。她透過畫菊來「勵人」，尤其是「吾輩女性」，表達在傳統社會框架下，女性仍能擁有「堅毅、不讓鬚眉」的氣概，具有個人修養與性別意識。她的志向是透過藝術創作，展現女性的才華與精神獨立。例如：「冀其能傳達吾輩女性之堅毅，不讓鬚眉之氣概。」直接點出為女性發聲的意圖。

3. 語言風格與情感表達：

  • 《尋熊記》 (黃美秀)：語言風格較為直接、理性，但又不失溫暖與真誠。情感表達是內斂而深沉的，透過對黑熊命運的關懷，自然流露出對生命的尊重與對環境的憂慮。例如：「我常常在想...」開篇即以第一人稱的思考，拉近與讀者的距離，傳達其真摯情感。
  • 《畫菊自序》 (張李德和)：語言風格較為典雅、含蓄，富有古典美。情感表達是藉由對菊花的描寫間接抒發，將個人情志寄託於物象之中，展現一種內斂而堅定的情懷。例如：「余雖一介女流，然志存高遠，不甘隨波逐流。」此句雖直接表志，但整體仍透過詠物來烘托。

總結：

黃美秀的筆法更具現代性與科學性，直接從生態問題出發，呼籲具體行動；張李德和的筆法則延續古典傳統，藉由詠物來含蓄地表達個人在特定時代背景下的自我期許與女性意識。兩者雖表達方式不同，但都展現了作者對其所關懷事物或自身價值的深刻投入與執著。
